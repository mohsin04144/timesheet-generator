How to Use — Timesheet Generator

1

Fill in Basic Info

Enter your Full Name, select your Project Team, and pick the Month & Year for the timesheet.
In Activity / Efforts, type the release or project you worked on (e.g. ServiceNow Release 25.1). Skip this if you use Work Phases.

Tip: Changing the month clears all leave and adhoc entries — set month first before adding other details.
2

Work Phases (Optional)

Toggle on if you worked on multiple releases within the month (e.g. Release 25.1 for the first half, 25.2 for the second half).
Set a From and To date for each phase and give it a label. The To date of one phase auto-snaps the From of the next.

Tip: Leave the Activity / Efforts field blank when using phases — each phase has its own label.
3

Daily Patterns

Your team's default pattern is pre-loaded (fixed meetings + adjustable main task). Each working day is filled using this pattern — 480 mins total per day.
Toggle Use custom patterns to override with your own rows. One row must be marked ADJ (adjustable) — it fills the remaining time.

Tip: Fixed rows run every day as-is. The ADJ row absorbs whatever time is left after fixed rows.
4

Leave Days (Optional)

Toggle on if you took any leave. Pick one or more dates from the calendar, choose the Leave Type (Sick / Planned / Comp Off / Holiday), then click Add Leave.
Repeat for different leave types. Leave days show as Other - Time Off rows in the final sheet.

Tip: Weekends are disabled. You cannot add adhoc work on a leave day.
5

Adhoc Efforts (Optional)

Toggle on to log extra work on specific days — such as a training session or a special call — on top of your daily pattern.
Each entry needs a date, effort type, effort details, and duration in minutes.

Tip: Adhoc minutes are added first; the ADJ pattern row fills the remainder. Total per day cannot exceed 480 mins.
6

Generate & Download

Click Generate Timesheet to preview all rows. Review the breakdown, then click Download .xlsx to save the file — ready to submit.
Use Copy Summary to copy a markdown summary to your clipboard for sharing in chat or docs.

Tip: The file is generated entirely in your browser — nothing is sent to any server.
