📋 How to Use — Timesheet Generator

⚡ What does this tool do?
It builds your monthly timesheet Excel file automatically. You fill in your details once, click Generate, and download a ready-to-submit .xlsx file. No manual row-by-row entry needed.
All 12 columns are filled automatically: LOB Vertical · Project Team · Effort Type · Associate Name · Date · Effort Details · Time Spent · Ticket Type · Ticket No. · SLA Met · Job Accuracy · Comments
1

Fill in Basic Info

This is the only required section. Fill in 5 fields:

▶
LOB Vertical — Your line of business. Pick from the list (SMT, SDLT, etc.) or choose "Other" to type your own.
▶
Associate Name — Your full name exactly as it should appear in the timesheet (e.g. John Smith).
▶
Project Team — Select your team. Named teams (SMT-Quality Management, SMT-ITSMRemedy, SMT-TMS) auto-load the correct daily pattern for you. Pick "Other" to type a custom team name and build your own pattern.
▶
Activity / Efforts — The release or project label (e.g. ServiceNow Release 25.3). Leave blank → shows as "General Activities". Not needed if using Work Phases.
▶
Month & Year — The month you are filing for. The tool generates rows for every Monday–Friday in that month. Weekends are skipped automatically.
⚠ Important: Set Month & Year first, before adding leave or adhoc entries. Changing them later will clear those entries.
1b

Work Phases (Optional)

Only needed if you worked on different releases or activities in the same month. For example: Release 25.1 for the first two weeks, then Release 25.2 for the rest of the month.

How to add a phase:

  1. Toggle on "I worked on multiple releases"
  2. Click + Add Phase
  3. Pick a From date and To date from the calendar
  4. Type the release / activity label for that date range
  5. Click Save — the next phase auto-starts from the next workday
Tip: You can have up to 4 phases. Days not covered by any phase fall back to the default Activity label you set in Basic Info.
2

Daily Patterns

A pattern is the set of rows that repeat every working day (e.g. Meetings 60 min, Development 360 min). For named teams, the correct pattern is loaded automatically — most people don't need to change anything here.

Default patterns per team:

SMT-Quality Management: Release status call (60m) + Daily Standup (60m) + Main task (360m ADJ)
SMT-ITSMRemedy: Daily Standup (60m) + Team Meeting (60m) + Main task (360m ADJ)
SMT-TMS: Dev Meeting (30m) + Staff Meeting (30m) + Misc (60m) + Main task (360m ADJ)

To customise: toggle on "Use custom patterns". Each row needs an Effort Type, Effort Details, Minutes, and type:

FIXED — same minutes every day ADJ — fills the remaining time to reach 480m (exactly one required)
3

Leave Days (Optional)

Toggle on if you had any days off. Leave days replace the full day's pattern with a single 480-minute row.

How to add leave:

  1. Toggle on "I had leave days this month"
  2. Click the date box — a calendar opens showing only your selected month
  3. Click one or more dates (weekends are greyed out)
  4. Choose the Leave Type from the dropdown
  5. Click + Add Leaves — dates appear as coloured chips
  6. Click ✕ on any chip to remove that date
Sick / Planned / Comp Off → Effort Type: Other - Time Off
Public Holiday → Effort Type: Other - Public Holiday
4

Adhoc Efforts (Optional)

For one-off extra work on a specific day — e.g. a training session, special call, or unplanned task — on top of your normal pattern.

How to add an adhoc entry:

  1. Toggle on "I have extra efforts on specific days"
  2. Click + Add Adhoc Entry — a row opens
  3. Pick a date (leave days are greyed out)
  4. Select Effort Type, enter Effort Details, set Minutes
  5. Click Save to confirm
How it works: Adhoc minutes are deducted from the ADJ row for that day, so the day still totals 480 minutes. For example, if you add 60m of adhoc training, the ADJ row reduces by 60m.
